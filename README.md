# Meal Search App

A simple web application to search for meals and view recipe details.

The app allows users to search meals by name, display results in cards, and view meal details inside a modal.

## Features

- Search meals by name
- Fetch data from external API
- Display meals in responsive cards
- Show recipe details in a modal
- Handle empty results and errors

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Font Awesome
- Forkify API

## How to Run

Open the `index.html` file in your browser.
